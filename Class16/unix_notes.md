## Core Unix cmds

Most unix cmds have super short names, which makes them quick to type but annoying to learn. Major file system related commands include

pwd: print working directory
ls: list files and directories
cd: change directory
mkdir: make a new directory
rm: remove files and directories (delete)
cp: copy files and dirs (source > destination)
mv: move files or dirs (basically re-name)
nano: a text editor (very basic but always available
